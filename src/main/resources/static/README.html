Date and time service

Summary:

This application is a RESTful web API which is based on java springboot framework. It takes two datetime as inputs (startDate and endDate) and based on the request it does respective calculations. Every datetime calculations has a two separate requests one that deals with input params which takes timezone offset and the other does not.

Pre-requisites:

Ensure you have the following installed and all respective env variables are set

  • Java 8+
  • GIT
  • maven 3+

Setup:

To run this application locally we need to do the following steps give below

  • Download the project folder in the local machine: git clone https://github.com/bhavanajrao/datetime_calculate_service.git
  • cd datetime_calculate_service
  • Set JAVA_HOME to java8: export JAVA_HOME {java_8_folder_path}
  • Download all the needed libraries: mvn clean package
  • Go to target folder: cd ./target
  • run the springboot jar file: java -jar spring-boot-0.0.1-SNAPSHOT.jar

Details of the Application:

Once the Application is UP and running we can send different requests which is explained in detailed below:

  • Calculate the Number of days between two Datetime: With This API you can calculate the number of days between two datetime which is passed as request params(startDate and endDate)
    • REQUEST URL: http://{hostname}:{port}/daysBetween
    • REQUEST PARAM 1 : startDate - pass a start datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST PARAM 2 : endDate - pass an end datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST: http://{hostname}:{port}/daysBetween?startDate={start_date_in_yyyy-mm-ddThh:mm:ss_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss_format}
    • EXAMPLE:http://localhost:8080/daysBetween?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31

  • Calculate the Number of days between two Datetime with timezone offset :With This API you can calculate the number of days between two datetime with timezone offset which is passed as request params(startDate and endDate)
    • REQUEST URL: http://{hostname}:{port}/daysBetween/withTimezones
    • REQUEST PARAM 1 : startDate - pass a start datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST PARAM 2 : endDate - pass an end datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST: http://{hostname}:{port}/daysBetween?startDate={start_date_in_yyyy-mm-ddThh:mm:ss[+|-]hh:mm_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss[+|-]hh:mm_format}
    • EXAMPLE:http://localhost:8080/weeksBetween/withTimezones?startDate=2021-12-01T06:40:31-05:30&endDate=2022-03-01T06:40:31+05:30

  • Calculate the Number of weekdays between two Datetime: With This API you can calculate the number of weekdays (i.e days from Monday to friday) between two datetime which is passed as request params(startDate and endDate)
    • REQUEST URL: http://{hostname}:{port}/weekdaysBetween
    • REQUEST PARAM 1 : startDate - pass a start datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST PARAM 2 : endDate - pass an end datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST: http://{hostname}:{port}/weekdaysBetween?startDate={start_date_in_yyyy-mm-ddThh:mm:ss_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss_format}
    • EXAMPLE:http://localhost:8080/daysBetween?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31

  • Calculate the Number of weekdays between two Datetime with timezone offset : With This API you can calculate the number of weekdays (i.e days from Monday to friday) between two datetime with timezone offset which is passed as request params(startDate and endDate)
    • REQUEST URI: http://{hostname}:{port}/weekdaysBetween/withTimezones
    • REQUEST PARAM 1 : startDate - pass a start datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST PARAM 2 : endDate - pass an end datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST: http://{hostname}:{port}/weekdaysBetween?startDate={start_date_in_yyyy-mm-ddThh:mm:ss[+|-]hh:mm_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss[+|-]hh:mm_format}
    • EXAMPLE:http://localhost:8080/weeksBetween/withTimezones?startDate=2021-12-01T06:40:31-05:30&endDate=2022-03-01T06:40:31+05:30

  • Calculate the Number of weeks between two Datetime: With This API you can calculate the number of full weeks between two datetime (i.e 7 days from the start date) which is passed as request params(startDate and endDate)
    • REQUEST URI: http://{hostname}:{port}/weeksBetween
    • REQUEST PARAM 1 : startDate - pass a start datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST PARAM 2 : endDate - pass an end datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST: http://{hostname}:{port}/weekBetween?startDate={start_date_in_yyyy-mm-ddThh:mm:ss_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss_format}
    • EXAMPLE:http://localhost:8080/daysBetween?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31

  • Calculate the Number of weeks between two Datetime with timezone offset : With This API you can calculate the number of weeks between two datetime with timezone offset (i.e 7 days from the start date) which is passed as request params(startDate and endDate)
    • REQUEST URI: http://{hostname}:{port}/weeksBetween/withTimezones
    • REQUEST PARAM 1 : startDate - pass a start datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST PARAM 2 : endDate - pass an end datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST: http://{hostname}:{port}/weeksBetween?startDate={start_date_in_yyyy-mm-ddThh:mm:ss[+|-]hh:mm_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss[+|-]hh:mm_format}
    • EXAMPLE:http://localhost:8080/weeksBetween/withTimezones?startDate=2021-12-01T06:40:31-05:30&endDate=2022-03-01T06:40:31+05:30

  • Calculate Number of days between two datetime and then convert it to either seconds, minutes, hours or years: With This API you can calculate the number of days between two datetime which is passed as reuest params(startDate and endDate) and then pass one more request param convertTo to convert the number of days into seconds, minutes, hours or years between the two datetime
    • REQUEST URI: http://{hostname}:{port}/daysBetween/convertedTo
    • REUEST PARAM 1 : startDate - pass a start datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST PARAM 2 : endDate - pass an end datetime in yyyy-mm-ddThh:mm:ss format,
    • REQUEST PARAM 3: convertTo - pass a unit of time you want to convert Number of days to, it can be either seconds, minutes, hours or years
    • REQUEST: http://{hostname}:{port}/daysBetween/convertedTo?startDate={start_date_in_yyyy-mm-ddThh:mm:ss_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss_format}&convertTo=seconds|years|minutes|years
    • EXAMPLE 1 - Convert number of days between the datetime to seconds :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=seconds
    • EXAMPLE 2 - Convert number of days between the datetime to minutes :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=minutes
    • EXAMPLE 2 - Convert number of days between the datetime to hours :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=hours
    • EXAMPLE 2 - Convert number of days between the datetime to years :http://localhost:8080/daysBetween/convertedTo?startDate=2020-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=years

  • Calculate Number of days between two datetime with timezone offset and then convert it to either seconds, minutes, hours or years: With This API you can calculate the number of days between two datetime with timezone offset which is passed as request params(startDate and endDate) and then pass one more request param convertTo to convert the number of days into seconds, minutes, hours or years between the two datetime
    • REQUEST URI: http://{hostname}:{port}/daysBetween/convertedTo
    • REUEST PARAM 1 : startDate - pass a start datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST PARAM 2 : endDate - pass an end datetime with timezone offset in yyyy-mm-ddThh:mm:ss[+|-]hh:mm format,
    • REQUEST PARAM 3: convertTo - pass a unit of time you want to convert Number of days to, it can be either seconds, minutes, hours or years
    • REQUEST: http://{hostname}:{port}/daysBetween/convertedTo?startDate={start_date_in_yyyy-mm-ddThh:mm:ss_format}&endDate={end_date_in_yyyy-mm-ddThh:mm:ss_format}&convertTo=seconds|years|minutes|years
    • EXAMPLE 1 - Convert number of days between the datetime to seconds :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=seconds
    • EXAMPLE 2 - Convert number of days between the datetime to minutes :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=minutes
    • EXAMPLE 2 - Convert number of days between the datetime to hours :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=hours
    • EXAMPLE 2 - Convert number of days between the datetime to years :http://localhost:8080/daysBetween/convertedTo?startDate=2021-12-01T06:40:31&endDate=2022-03-01T06:40:31&convertTo=years
